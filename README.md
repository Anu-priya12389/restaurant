# 🍽️ A Table For Two - Restaurant Website :  https://anu-priya12389.github.io/restaurant/

## 📌 Project Description

**A Table For Two** is a romantic restaurant website designed especially for couples who want to enjoy delicious food and meaningful moments together.

The website presents a warm and elegant dining experience through a visually appealing interface. It includes a food menu, different meal categories, a price list, restaurant information, and contact details.

> **Where the hearts and plates connect...** ❤️
